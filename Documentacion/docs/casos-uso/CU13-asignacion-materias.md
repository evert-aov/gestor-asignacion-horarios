# CU13 - Asignar Materias a Docentes

| Campo             | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Caso de uso**   | CU13 - Asignar Materias a Docentes                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Propósito**     | Permite al Administrador gestionar las materias que imparte cada docente, incluyendo la asignación de materias con sus respectivas carreras universitarias.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Actores**       | • Administrador                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Iniciador**     | • Administrador                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Precondición**  | El Administrador debe haber iniciado sesión en el sistema (CU01).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Flujo**         | • El Administrador accede al módulo de "Procesos Académicos" → "Teacher Subject Manager".<br>• El sistema muestra la lista de docentes activos.<br>• El Administrador puede buscar un docente específico por nombre, apellido, email o número de documento.<br>• El Administrador selecciona un docente para editar sus materias asignadas.<br>• El sistema muestra un modal con:<br>&nbsp;&nbsp;- Información del docente seleccionado<br>&nbsp;&nbsp;- Lista de materias disponibles<br>&nbsp;&nbsp;- Materias actualmente asignadas al docente<br>&nbsp;&nbsp;- Carrera universitaria asociada a cada materia<br>• El Administrador puede:<br>&nbsp;&nbsp;- Agregar nuevas materias al docente<br>&nbsp;&nbsp;- Asignar la carrera correspondiente a cada materia<br>&nbsp;&nbsp;- Eliminar materias previamente asignadas<br>• El sistema valida que:<br>&nbsp;&nbsp;- Se seleccione al menos una materia<br>&nbsp;&nbsp;- El usuario sea un docente válido<br>&nbsp;&nbsp;- Las materias seleccionadas existan y estén activas<br>• El sistema guarda los cambios en la base de datos usando transacciones.<br>• El sistema registra las acciones en la auditoría. |
| **Postcondición** | Las materias quedan asignadas al docente y disponibles para su uso en la asignación de carga horaria. Las acciones quedan registradas en la auditoría.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Excepción**     | • No se selecciona ninguna materia.<br>• El usuario seleccionado no es un docente válido.<br>• Error en la transacción de base de datos.<br>• Se ingresan datos inválidos o incompletos.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |

## Validaciones Implementadas

1. **Validación de Docente**: Verifica que el usuario tenga el rol "Docente".
2. **Validación de Materias**: Verifica que se seleccione al menos una materia válida.
3. **Validación de Datos**: Filtra valores únicos y válidos antes de guardar.
